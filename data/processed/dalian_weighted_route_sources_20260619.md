# 大连 3 天 2 晚权重版路线与引用摘要（2026-06-19）

## 最新权重

- 拍照：40%
- 好吃：35%
- 顺路：15%
- 不累：10%

处理原则：
- 东港音乐喷泉是今晚核心意图，因此 Day1 从“东港夜景”升级为“东港喷泉优先、威尼斯水城兜底”。
- 渔人码头保留为 Day1 时间充足时的可选拍照点；若酒店出发晚，直接跳过，保证喷泉。
- 餐饮推荐不只看热度，也看多条个人笔记是否交叉提到，以及是否顺路、是否存在排队/口味分歧。

## 当前主线

Day 1：酒店 -> 中山广场 -> 可选渔人码头 -> 港东五街 -> 东港音乐喷泉广场 -> 东港威尼斯水城 -> 东港晚餐

Day 2：酒店 -> 白云山/鹿 -> 莲花山 -> 星海广场 -> 午餐/休息 -> 小付家庄 -> 傅家庄 -> 银沙滩 -> 新长兴/西安路晚餐

Day 3：酒店退房 -> 七一早市 -> 高尔基街/大连美术馆 -> 东关街 -> 南山 -> 正黄旗 -> 不咸糖水 -> 大连商场 -> 回酒店取行李 -> 大连北站

- 东港音乐喷泉没有稳定公开官方日历，本轮按用户样本和当天体验意图安排；必须以现场/高德/当天公告为准。
- 餐饮全部按个人笔记交叉判断，不视为广告背书；有评价分歧的店只放备选或“排队短再吃”。
